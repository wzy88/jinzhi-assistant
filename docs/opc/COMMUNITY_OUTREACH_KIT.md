# 社区触达材料

## 1. 微信首条消息

```text
您好，我正在参加天津市首届 AIGC 应用创新大赛，项目方向是“社区网格 AI 助理”。

我做了一个很轻量的小工具：网格员把居民诉求粘贴进去后，系统会自动判断类别、紧急程度、情绪标签、建议处理部门，并生成处理建议和居民通知。

现在不需要接入任何真实系统，也不收集个人隐私。我只想请您帮忙看一下这个工具是否贴近日常工作，或者提供几条脱敏后的常见诉求样例做测试。

如果您方便，我可以用 15 分钟线上演示一下。后续如果您觉得有价值，也可以免费开放给社区试用 1-3 个月。
```

## 2. 电话沟通话术

```text
您好，我是参加天津 AIGC 应用创新大赛的选手，现在做的项目是社区网格 AI 助理。

它不是要替代社区人员，而是帮网格员先把居民诉求整理成结构化工单，比如自动判断是消防安全、环境卫生、设施维修还是邻里纠纷，同时生成摘要、建议处理步骤和居民通知。

我们现在只是做轻量试点，不需要接入任何系统，也不需要真实隐私数据。您只要给几条脱敏后的常见诉求，或者听我演示 15 分钟，帮忙判断有没有用就可以。

如果后续觉得有价值，我们可以免费试用一段时间，再看是否继续。
```

## 3. 访谈邀约

```text
想请您帮忙做一个 15 分钟的小访谈，主要了解：

1. 社区/物业平时怎么接收居民诉求。
2. 哪些问题最耗时间。
3. AI 自动分类、摘要、通知生成是否有帮助。
4. 如果试用，需要满足哪些隐私和管理要求。

我们会把所有内容脱敏，只用于比赛项目验证，不公开具体个人信息。
```

## 4. 请求脱敏样例

```text
如果方便的话，能否提供 5-10 条脱敏后的常见诉求样例？

格式可以很简单，每行一句话即可。例如：
“某楼栋消防通道被占用，希望社区协调。”
“小区垃圾桶满了两天没人清理，味道很大。”

请不要包含居民姓名、手机号、身份证、具体门牌号、车牌号等隐私信息。
```

## 5. 试用反馈请求

```text
您刚才试用后，能否帮忙简单反馈三点：

1. 分类和紧急度判断是否基本准确？
2. 通知文案是否能直接或稍微修改后使用？
3. 是否愿意继续试用，或者推荐给其他社区/物业同事看看？

一句话反馈也可以，我们会整理成脱敏的试点验证材料。
```

## 6. 试用确认文本

```text
我已体验“津智助理”社区网格 AI 工单预处理工具。该工具可对居民诉求进行分类、紧急度判断、摘要生成和通知生成。初步体验认为该方向对社区/物业工单整理有一定参考价值，建议后续结合脱敏真实样例继续试用和优化。
```

## 7. 可接受的替代试点

如果暂时找不到社区，可联系：

- 物业客服。
- 高校后勤。
- 宿舍报修中心。
- 园区物业。
- 街道活动组织人员。
- 学院办公室。

表达时统一归为：

> 基层服务工单预处理和通知生成场景。

## 9. 注意事项

1. 不承诺替代人工判断。
2. 不收集个人隐私。
3. 不声称已有真实试点，除非已经拿到。
4. 不把模拟数据说成真实数据。
5. 重点强调免费、轻量、脱敏、可随时停止。
