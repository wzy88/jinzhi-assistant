# 津智助理 OPC 项目包

这组文档用于证明：津智助理不是只看完比赛文档后做出的网页 Demo，而是一个由单人借助 AI 完成的、从立项到交付使用的完整 OPC 项目。

OPC 参赛叙事应围绕一条主线展开：

> 我作为一个超级个体，用 AI 工具完成了社区网格智能助手从市场机会识别、需求访谈、产品设计、开发测试、试点交付、反馈复盘到商业化设计的全过程。

## 3. 初赛提交组合

初赛不要只提交网站链接。建议提交：

1. 产品链接：`https://jinzhi.22dhmv.top`
2. 1 分钟演示视频
3. 产品介绍 PDF
4. 本 OPC 项目包摘要
5. 脱敏样例数据与准确率评测截图
6. 访谈记录或试用意向截图

## 4. 路演一句话

津智助理把居民诉求自动转化成可分类、可分派、可回复、可复盘的社区工单；更重要的是，它展示了一个人如何用 AI 在 90 天内完成从政务场景洞察到可交付 SaaS 原型的完整创业闭环。
